# League of Legends College Website

A fan-made League of Legends website built as a mini project (CP 3620).

## Pages

- **PV Page** — Landing page with a promotional video
- **Home** — Overview of game features, champions, and highlights
- **Champions** — Champion list with individual detail pages (Ahri, Akali, Alistar, Ashe, Darius, Janna)
- **Download** — Download page
- **Sign Up / Log In** — User authentication pages

## Tech Stack

- HTML, CSS, JavaScript
- Static website deployed via GitHub Pages

## Reference

Design and content referenced from the official [League of Legends website](https://www.leagueoflegends.com/en-us/).
